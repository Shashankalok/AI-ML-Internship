# 🚀 LangChain Basics Projects

This repository contains hands-on projects built while learning the **LangChain Framework**.
The goal is to understand how to build real-world AI applications using LLMs.

---

## 📚 What is LangChain?

LangChain is a framework used to build applications powered by Large Language Models (LLMs).
It helps in combining prompts, memory, tools, and workflows to create intelligent systems.

---

## 🧠 Key Concepts Learned

### 🔹 Prompt Templates

Prompt templates allow dynamic input to LLMs using variables.

### 🔹 LLM Chains

Chains combine prompts and models to create structured workflows.

### 🔹 Sequential Chains

Sequential chains execute multiple steps where the output of one step becomes the input of the next.

### 🔹 Memory

Memory enables conversational AI by storing previous interactions.

### 🔹 Output Parsing

Structured outputs (like JSON) can be extracted from unstructured text.
